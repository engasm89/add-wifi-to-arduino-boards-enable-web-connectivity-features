## Connections

- WiFi module (ESP8266/ESP01): `TX -> RX`, `RX -> TX (via divider)`, `VCC -> 3.3V`, `GND -> GND`.

## Diagram (ASCII)

```
 RX <- TX Module   TX -> RX Module (level shift)
 3V3/GND -> Module power
```
